# Reference
[typedef_specifier](https://en.cppreference.com/w/cpp/language/typedef)

Typically, the `typedef` specifier appears at the start of the declaration, though it is permitted to appear after the type specifiers, or between two type specifier.

> The typedef-names are alias for existing types, and are not declarations of new types. 
> Typedef names are only in effect in the scope where they are visible.
