# The basic

- the `start` and `end` character should be `/` (frontward **slash**)
- the character behind end `/` will be `Flag`, the types as following
  - `g`lobal
  - case `i`nsensitive
  - `m`ultiline
  - `s`ingle line (dotall)
  - `u`nicode
  - stick`y`
